rep

 

Description

 

Obtain g-blink representant.

 

Examples

 

Usage

 

rep(gblink,D=1,F=0,DF=0)

 

     Return the representant considering dualization of each block if D=1 (default), reflection of each block if F=1 (not default) and refDual of each blink if DF=1 (not default).

 

Arguments

 

gblink

The gblink to draw  

D=1

Consider the dual of each block  

R=0

Consider the reflection of each block

RD=0

Consider the refdual of each block
