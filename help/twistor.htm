twistor

 

Description

 

List the twistors of a 3-gem or apply a twisting on a 3-gem resulting in a new 3-gem.

 

Examples

 

Usage

 

twistor(gem)

 

Lists all twistors of the given gem.

 

twistor(gem,c1,c2,v1,v2)

 

Returns a new 3-gem obtained from the given gem by applying a (c1,c2)-twist at a c2-twistor on (v1,v2) forming a c3-twistor, where {c1,c2,c3} = {1,2,3} and the parity of v1 is the same as of v2. A (c1,c2)-twist is defined by interchanging the 0 and c1 neighbors of vertices v1 and v2.

 

Arguments

 

gem

3-gem for the list of twistors or the base of application of the twisting operation.  

c1

Label of a color in {1,2,3}

c2

Label of a color in {1,2,3}

v1

Label of vertex v1

v2

Label of vertex v2
