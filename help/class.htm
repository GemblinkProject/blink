class

 

Description

 

Returns a class of g-blinks.

 

Examples

 

Usage

 

class()

 

Returns all classes on census with id=1

 

class(4,1)

 

Returns the first class with 4 edges and order 1 in the census with id=1

 

class(4,1,id=2)

 

Returns the first class with 4 edges and order 1 in the census with id=2

 

class(qi=1)

 

Returns the classes with integral quantum invariant up to its rmax. The convention is 1 = integral, 2 = real, 4 = complex. Sum to get a combination. The default is qi=7 (all).
