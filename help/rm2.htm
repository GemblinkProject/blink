rm2

 

Description

 

List all the Reidemeister Move II points of application on a given g-blink or apply such a move in a given gblink and application point. 

 

 

Examples

 

Usage

 

rm2(gblink)

 

Lists all application points of Reidemeister Move III on the given g-blink.

 

rm2(gblink,a,t1)

 

Apply rm2 on hexagon starting in vertex label a and ab is connected by t1 in {“v”, “f”} indicating that t1 is a vertex-edge or a face-edge.

 

Arguments

 

gem

3-gem for the list of twistors or the base of application of the twisting operation.  

c1

Label of a color in {1,2,3}

c2

Label of a color in {1,2,3}

v1

Label of vertex v1

v2

Label of vertex v2
