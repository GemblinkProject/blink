drawcp

 

Description

 

Draw, from a gblink, a link or/and a blink using the Circle Pack algorithm.

 

Examples

 

Usage

 

drawcp(gblink,L=1,B=0,C=0,w=500,h=500)

 

Arguments

 

gblink

The gblink to draw  

L=1

Draw link  

B=0

Draw blink

C=0

Draw circles

w=500

Width of image

h=500

Height of image
